# Evaluation Outputs

This directory contains evaluation results from the MLOps Session 29 pipeline.

## Structure
- `static/` - Static evaluation results (test suite results)
- `dynamic/` - Dynamic evaluation results (human reviews)
- `hybrid/` - Hybrid evaluation results (Promptfoo, OpenAI Evals)
- `openai_evals/` - OpenAI Evals specific results
- `ci_cd/` - CI/CD workflow configurations

## Note
Large result files (.json, .log) are git-ignored.
Keep only summary reports and important artifacts in version control.
